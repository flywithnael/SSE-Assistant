Bienvenue sur le site officiel de SSE Assistant

SSE Assistant est un logiciel créé pour améliorer l'expérience des conducteurs dans les réseaux virtuels, avec une interface intuitive et utile.

Fonctionnalités

  • Affichage des arrêts par ligne
  • Commandes clavier associées
  • Support SAE
  • Mode simplifié
  • Personnalisation complète
